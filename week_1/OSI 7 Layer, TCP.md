## **OSI 7 Layer**

- 7 계층 - Application layer
- 6 계층 - Presentation layer
- 5 계층 - Session layer
- 4 계층 - Transport layer
- 3 계층 - Network layer
- 2 계층 - Link layer
- 1 계층 - Physical layer

## **TCP/IP**

- 4 계층 - Application layer
- 3 계층 - Transport layer
- 2 계층 - Internet layer
- 1 계층 - Network Access layer
